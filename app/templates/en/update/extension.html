New to MarketoLive Extension v5.0.0

Extension is Out of Date Notification



If you receive this notification you will not be able to use the MarketoLive instance until you update the extension. Simply click on the 'Update Extension' button to open the Chrome Extensions page in order to resolve the issue.

Extension Update Pending Notification



In the future you will receive this message anytime a new extension update is available. Simply click on the 'Approve Update', button to apply the update if it doesn't require any permission changes, otherwise you will need to manually approve the update.

Bad Extension Notification



If you receive this notification you will not be able to use the MarketoLive instance until the bad extension is uninstalled. Simply click on the 'Uninstall Bad Extension' button to resolve the issue.

New to MarketoLive Instance

Email Insights Update



You can now demo live data within Email Insights rather than the previous static app. If you would still like to demo the static app, please use the link below.
http://www.marketolive.com/en/analytics/email-insights-summit-demo-1

Saving & Approving Landing Page Templates



Due to an bug with how Landing Page Templates load in the editor, the only way you can now Save or Approve templates within your user folder in My Workspace is to first Preview the template then switch to the editor via the Edit Draft toolbar button in the previewer.

Coming Soon to MarketoLive

Update Grace Period



There will be a grace period of 1 week in which to update the extension to the latest version (5.0.0), after which any user who does not have the latest version installed will not be able to use the MarketoLive instance and will receive the dialog window above.

MarketoLive Reloaded



In the coming weeks the Demo Services Team will be providing access to the next generation demo environment, code named MarketoLive Reloaded, to Solution Consultants.

This demo environment will enable you to demo the full capabilities of our solutions with real and organic demo data, no more static data or dummy dashboard data. It will enable you to demo in the way that is best for you and your deal, either from a functionality or journey perspective. Additionally, you will be able to demo the entire Admin section safely.
