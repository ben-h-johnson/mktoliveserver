Latest MarketoLive Extension v5.1.6

New to MarketoLive Extension

Minimum Required Version





We are now enforcing a minimium required version of 5.0.0, any user who has an older version installed will not be able to use the MarketoLive instance and will receive the dialog window and the Chrome notification shown above.

Clear Your Cache Easily



Fix basic issues by clearing your cache easily in 3 simple clicks:
1. Click the extension icon in the Chrome toolbar
2. Click the gear icon in the extension popup window
3. Click the CLEAR CACHE button

It will automatically clear all your cached images and files from the beginning of time, reload all Marketo or MarketoLive tabs, and then close the popup window.
This will NOT clear your cookies, browsing history, or autofill form data.

Changes Are Not Allowed Notification



You will receive this notification if you attempt to make changes to the demo environment that aren't allowed, the page will reload automatically as a result.

Bad Extension Notification



If you receive this notification you will not be able to use the MarketoLive instance until the bad extension is uninstalled. Simply click on the 'Uninstall Bad Extension' button to resolve the issue.

Saving & Approving Landing Page Templates



Due to an bug with how Landing Page Templates load in the editor, the only way you can now Save or Approve templates within your user folder in My Workspace is to first Preview the template then switch to the editor via the Edit Draft toolbar button in the previewer.

MarketoLive Reloaded



All Solutions Consultants now have access to the next generation demo environment, code named MarketoLive Reloaded.

To login, your user ID is {email}@marketolive.com (e.g. bfisher@marketo.com --> bfisher@marketolive.com)

This demo environment will enable you to demo the full capabilities of our solutions with real and organic demo data, no more static data or dummy dashboard data. It will enable you to demo in the way that is best for you and your deal, either from a functionality or journey perspective. Additionally, you will be able to demo the entire Admin section safely.
