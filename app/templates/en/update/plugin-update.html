Performance Enhancements

What is Changing?

In the past the extension loaded many scripts onto any Marketo instance page which impacted browser performance, NOT page load time. As a result, it loaded the scripts for populating the various dashboards with our demo data which created much confusion with CAMs in particular who on occasion may log into their customer’s instance. Now the extension loads the bulk of the scripts on only our MarketoLive instance but it will still provide the dashboard demo data and deep links for the Email Insights and Email Deliverability Tools tiles within all demo instances (any instance whose account string starts with mktodemoaccount). This should also help to minimize seeing the Demo Plugin Warning window on login for those people who may access MarketoLive on a slower internet connection.

Discarding Landing Page Drafts Enhancement

What is Changing?

In the past the extension would issue a single request to discard all of the gold standard Landing Pages, however if any Landing Page that was included in this request had been subsequently deleted, the entire request would fail leaving all drafts not discarded. Now the extension issues a single request for each of the gold standard Landing Pages such that if one Landing Pages gets subsequently deleted the others will still get successfully discarded.
